# 🎮 Memory Card Game

A fun and interactive **Memory Matching Game** built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- 🎴 Flip & Match Cards
- ⏱ Timer System
- ⭐ Move Counter
- 🏆 Win Popup
- 🎚 Difficulty Levels
- 🌙 Neon Theme Mode
- 📱 Responsive Design

---

## 🧠 Tech Stack

- HTML5
- CSS3 (Glassmorphism UI)
- JavaScript (Game Logic)
